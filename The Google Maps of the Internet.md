## What is a Domain Name?
A domain name is a humanly understandable form of any web application hosted on one or more servers, and it helps us connect to them.

## What is DNS (Domain Name System)?
A DNS or Domain Name System is a system of devices that helps us find the corresponding IP address of a domain name.
