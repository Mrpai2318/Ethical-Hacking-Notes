## Some of the most commonly used Tags
1.  Header `<h1>`
  * Headers show large header text, usually used for a title of your page.
  * The number after `H` signifies the size of the header, 1 being the largest and 5 the smallest.

2. Nav `<nav>`
  * A nav element's purpose is to provide navigation links for the site
  * Usually positioned at the top of the page

3.  Paragraph `<p>`
  * Paragraph elements are for body text on a web page
  * A new paragraph element is necessary for new paragraphs rather than using the _return_ key

4.  Div `<div>`
  * Div tags are block elements that can be used to contain other html elements
  * You may notice divs are used for much more than that exploring HTML code throughout the web.

5.  Input `<input>`
  * Inputs are plain by themselves but using attributes can be modified to be a button, text input or more.
  * Other inputs include: checkbox, radio, text-area, select,

6.  Button `<button type="#">`
  * `<button type="submit">` submits a form after user has input information
  * `<button type="button">` bypasses HTML default submit functionality to make possible customize button interaction with JavaScript

7. Image `<img src="#">`
  * this tag allows you to place either URLs or a path to an image source to view on your webpag
